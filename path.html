$PATH

เส้นทาง ไว้สำหรับเซ้ตคำสั่งต่างๆ คำสั่งที่สร้างเอง มีในระบบแล้ว

ตัวแปร PATH จะเก็บตำแหน่งของ directory ต่าง ๆ

ยกตัวอย่างการใช้งาน

echo $PATH -------) /opt/local/bin:/usr/X11R6/bin:/usr/bin:/usr/sbin/:/bin

เราสามารถเพิ่ม path ของโปรแกรมเข้าไปที่ตัวแปร PATH ได้โดย เช่น เมื่อต้องการเพิ่ม path /home/top85/ns-allinone-2.31 เข้าไปในตัวแปร PATH ทำได้โดยใช้คำสั่ง export PATH="/home/top85/ns-allinone-2.31:$PATH"
